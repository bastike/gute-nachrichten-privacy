📰 Über die App & Quellen

„Gute Nachrichten" ist ein Nachrichten-Aggregator (Non-Profit-Projekt), der positive Nachrichten aus öffentlich zugänglichen RSS-Feeds sammelt und auf Deutsch bereitstellt.

Für jeden Artikel werden angegeben:

  • Name des Originalverlags (Quelle)
  • Veröffentlichungsdatum
  • Herkunftsland
  • Direkter Link zum Originalartikel

Es werden ausschließlich Artikel angezeigt, die weniger als 3 Monate alt sind. Die Inhalte werden regelmäßig automatisch aktualisiert.

🔒 Datenschutzerklärung

1. Verantwortlicher

Diese App wird als Non-Profit-Projekt betrieben.
Kontakt: bastikeidler@outlook.de

2. Welche Daten werden erhoben?

Die App „Gute Nachrichten" erhebt keine personenbezogenen Daten. Es werden weder Name, E-Mail-Adresse, Standort noch andere persönliche Informationen erfasst.

3. Lokale Datenspeicherung

Die App speichert ausschließlich lokal auf Ihrem Gerät:

  • Nachrichtenartikel aus öffentlichen RSS-Feeds (für die Anzeige)
  • Ihre Feed-Einstellungen (aktivierte/deaktivierte Feeds)
  • Übersetzungs-Cache (zur Verbesserung der Ladezeiten)

Diese Daten werden nicht an Server übertragen und verbleiben ausschließlich auf Ihrem Gerät.

4. Netzwerkzugriffe

Die App greift auf das Internet zu, um:

  • RSS-Feeds abzurufen: Öffentlich zugängliche Nachrichtenfeeds werden direkt von den jeweiligen Nachrichtenquellen geladen.
  • Feed-Liste zu laden: Die aktuelle Liste der RSS-Quellen wird von dieser Seite geladen.
  • Übersetzungen durchzuführen: Texte werden an kostenlose Übersetzungsdienste gesendet. Dabei werden keine Nutzerdaten übertragen, sondern ausschließlich der zu übersetzende Nachrichtentext.
  • Bilder zu laden: Nachrichtenbilder werden von den jeweiligen Quellseiten geladen.

5. Drittanbieter-Dienste

Die App nutzt folgende externe Dienste ausschließlich zur Textübersetzung:

  • Lingva Translate (Open Source Google-Translate-Proxy)
  • Google Translate (öffentliche API)
  • SimplyTranslate (Open Source)
  • MyMemory Translation API
  • LibreTranslate (Open Source)
  • Apertium (Open Source)

Es werden dabei keine Tracking-Cookies, Werbe-IDs oder Analytics-Daten verwendet.

6. Keine Werbung, kein Tracking

Die App enthält:

  • Keine Werbung
  • Kein Tracking oder Analytics
  • Keine Social-Media-SDKs
  • Keine Nutzerprofile
  • Keine Registrierung oder Anmeldung

7. Berechtigungen

Die App benötigt nur eine Berechtigung:

  • Internet (INTERNET): Zum Laden von RSS-Feeds und Übersetzungen

8. Datenlöschung

Alle lokal gespeicherten Daten können jederzeit gelöscht werden, indem Sie:

  • Die App deinstallieren, oder
  • Die App-Daten in den Android-Einstellungen löschen

9. Änderungen

Diese Datenschutzerklärung kann bei Bedarf aktualisiert werden. Änderungen werden in der App und auf dieser Seite veröffentlicht.
